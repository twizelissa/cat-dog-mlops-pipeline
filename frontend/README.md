# Cats vs Dogs Classifier - Next.js Frontend

Modern, responsive frontend for the ML model.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Dashboard with real-time metrics
- Single image prediction
- Bulk data upload and retraining
- Beautiful visualizations with Recharts
- Responsive design with Tailwind CSS

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Recharts
- Axios
